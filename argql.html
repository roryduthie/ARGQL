Search for arguments explicitly related with one of the available relations (rebut, undercut, attack, endorse, back, support).

Searching is performed also with equivalent propositions. In the example, proposition "every candidate is a broken record if you listen to them everyday" is equivalent with the "all candidates are broken record" which is the actual conclusion of the argument.

Query preview

match ?a: < ?pr1, "every candidate is a broken record if you listen to them everyday" >
return ?a

Expressing a path pattern as a sequence of relations in the graph of arguments

Query preview

The path*n numeric indicator in the path pattern implies n repetitions of the path

Query preview

match ?a (back)*5 ?b: < ?pr1, "Yet if the carriers are not built and properly equipped a shadow will fall over the whole of Britain's defence capability" >
return ?a, ?b

The path+n numeric indicator in the path pattern implies up to n repetitions of the path

Query preview

match ?a (rebut)+2 ?b
return ?a, ?b

A path pattern can be expressed as an arbitrary number of combinations of explicit path patterns and of paths with numerical indicators.

Query preview

match ?a ((rebut/undercut)*1)+2 ?b: < ?pr, "TRUMP's telling the truth" >
return ?a, ?b
